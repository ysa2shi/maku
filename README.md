# 📝 **MAKU** - GitHub プロフィール README ジェネレーター (開発中)

> ミニマリズムと間（ま）の美学に基づき、日本的で洗練されたGitHubプロフィールREADMEをGUIで簡単に生成できるツール。

## 🎯 コンセプト

**MAKU** は、簡潔な操作で作成できるサーバー駆動型ジェネレーターです。

- 🎨 **デザイン:** 胡粉（ごふん）と錆青磁（さびせいじ）を基調にした和の色彩体系
- ⚡ **技術:** Astro SSR + htmx による高速なサーバー駆動型UI
- 🌍 **対象:** 日本の開発者および繊細なデザインを好む国際的な開発者

---

## ✨ 主な機能

| 機能 | 詳細 |
| :--- | :--- |
| **基本情報入力** | GitHubユーザー名、自己紹介文、ソーシャルリンク |
| **スキル選択** | Skill Iconsを利用した技術アイコンの選択UI |
| **ダイナミックバッジ** | 訪問者数、GitHubトロフィー、統計、ストリーク情報 |
| **リアルタイムプレビュー** | 入力変更時にサーバーで生成されたHTMLをリアルタイムで表示 |
| **コピー機能** | 生成されたマークダウンをワンクリックでコピー |
| **言語切り替え** | ブラウザの`Accept-Language`に基づき自動判定、手動切り替え対応 |

---

## 🏗️ 技術スタック

| カテゴリ | 技術 | 役割 |
| :--- | :--- | :--- |
| **フレームワーク** | Astro (SSR) | サーバーサイドレンダリングとAPI Route |
| **通信/レンダリング** | htmx | サーバー駆動型UIの実現 |
| **クライアントロジック** | Alpine.js | 局所的なUI制御（絵文字パレット、コピー機能など） |
| **スタイリング** | Tailwind CSS | 「間（ま）の美学」に基づくデザイン |
| **ホスティング** | Cloudflare Pages | サーバーレス環境でのAstro SSR実行 |

## 🚀 セットアップと実行

### 必要な環境

- Node.js 18+ と pnpm

### インストール

```sh
pnpm install
```

### 開発サーバーの起動

```sh
pnpm dev
```

`localhost:4321` でアプリケーションが起動します。

### ビルド

```sh
pnpm build
```

### プレビュー

```sh
pnpm preview
```

## 🎨 デザイン原則

### 配色

| 色名 | HEXコード | 用途 |
| :--- | :--- | :--- |
| 胡粉（ごふん） | `#F9F9F4` | 背景 |
| 墨色（すみいろ） | `#222222` | メインテキスト |
| 錆青磁（さびせいじ） | `#669999` | アクセント色、フォーカス時の下線 |

### タイポグラフィ

- **見出し:** 明朝体 (`font-serif`)
- **本文・入力:** 細めのゴシック体 (`font-sans`)

### 「間（ま）」の美学

- 入力エリアとプレビューエリア間に大きな余白を配置
- 各コンポーネントのパディングを余裕を持って設定
- セクション区切りは濃い線ではなく極細の薄墨点線を使用
